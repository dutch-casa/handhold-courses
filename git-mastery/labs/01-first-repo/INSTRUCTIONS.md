# Lab 1: Your First Repository

Initialize a git repository and record your first commit.

## Tasks

- [ ] Initialize a git repository in the current directory with `git init`
- [ ] Stage `hello.txt` for commit with `git add`
- [ ] Commit the file with the message `first commit`

## Hints

- `git init` creates a `.git/` directory — that's the object store.
- `git add hello.txt` stages the file. `git add .` stages everything.
- `git commit -m "first commit"` records the snapshot.
- `git status` shows you what is staged, what is not, and whether the tree is clean.

Use **Run Tests** in the testing panel to check your progress.
