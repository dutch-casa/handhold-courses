# Test-Driven Add

Implement arithmetic functions to pass the provided tests.

## Tasks

- [ ] Implement `add(int, int)` that returns the sum of two integers
- [ ] Implement `subtract(int, int)` that returns the difference
- [ ] Implement `multiply(int, int)` that returns the product
- [ ] Implement `clamp(int value, int lo, int hi)` that returns value constrained to the range [lo, hi]

## Hints

Look at `test/test.cpp` to see exactly what each function must do.

`clamp` should return `lo` if value is below lo, `hi` if value is above hi, and `value` otherwise.

Use `Run Tests` in the testing panel to check your progress.
